# Claude Skills Library 使い方ガイド

## 概要
このリポジトリは、Claude/ChatGPT/AIを活用したスキル、プロンプト、ワークフローを管理するためのライブラリです。

## ディレクトリ構造
```
claude-skill/
├ skills/          # 個別のスキル定義
├ workflows/       # ワークフロー定義
├ templates/       # テンプレートファイル
└ docs/           # ドキュメント
```

## スキルの作成方法
1. `templates/skill-template.md` をコピー
2. スキルの目的、入力、出力、使用例を記述
3. 実際のプロンプトを記載
4. `skills/` フォルダに保存

## ワークフローの作成方法
1. `templates/workflow-template.md` をコピー
2. ワークフローの目的とステップを定義
3. 使用するツールと入出力を記述
4. `workflows/` フォルダに保存

## 使用例
### スキルの使用
1. 目的のスキルファイルを開く
2. プロンプト部分をコピー
3. Claude/ChatGPTに貼り付けて使用

### ワークフローの実行
1. ワークフローファイルを確認
2. 必要なツールを準備
3. ステップに従って実行

## ベストプラクティス
- スキルの目的を明確に定義する
- 入力と出力を具体的に記述する
- 使用例を豊富に記載する
- 定期的にスキルを更新・改善する
