# AIニュース → 要約 → X投稿 ワークフロー

## 概要
AI関連のニュースを収集し、要約を作成してXに投稿する自動化ワークフローです。情報発信の効率化を実現します。

## ステップ
1. ニュース収集: AI関連ニュースソースから最新記事を取得
2. 要約生成: AIニュース要約スキルを使用して記事を要約
3. 投稿作成: X投稿生成スキルを使用して投稿文を作成
4. 投稿実行: X APIを使用して投稿を自動実行

## 使用ツール
- RSSフィードリーダー: ニュース収集
- Claude API: 要約と投稿生成
- X API: 投稿実行
- スケジューラー: 定期実行（例: 毎日9時）

## 入力
- ニュースソース: 監視するRSSフィードURL
- 投稿頻度: 投稿する間隔（例: 1日1回）
- トーン設定: 投稿のトーン設定

## 出力
- 投稿履歴: 実行された投稿の記録
- エンゲージメント分析: 投稿の反応分析
- 要約アーカイブ: 生成された要約の保存

## 実行例
1. RSSから最新のAIニュースを取得
2. 各記事をClaudeで要約
3. 要約内容を基に魅力的なX投稿を作成
4. 自動的にXに投稿
