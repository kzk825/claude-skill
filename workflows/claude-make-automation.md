# Claude × Make Automation ワークフロー

## 概要
Claude AIとMake.comを連携させた自動化ワークフローです。Claudeの高度な分析能力をビジネスプロセスに統合します。

## ステップ
1. トリガー検知: Make.comで特定のイベントを検知（例: 新しい問い合わせ）
2. データ収集: 関連データの収集と整理
3. Claude分析: Claude APIでデータ分析と判断
4. アクション実行: 分析結果に基づく自動アクション
5. 結果通知: 実行結果の通知とログ記録

## 使用ツール
- Make.com: ワークフロー自動化プラットフォーム
- Claude API: AI分析と生成
- データベース: 情報の一時保存
- 通知サービス: 結果通知（Slack, Emailなど）

## 入力
- トリガー条件: ワークフローを開始する条件
- 分析タスク: Claudeに依頼する分析内容
- アクション設定: 実行する自動アクション

## 出力
- 分析結果: Claudeからの分析レポート
- 実行ログ: 各ステップの実行結果
- パフォーマンス指標: ワークフローの効率性分析

## 実行例
1. 新しい顧客問い合わせを検知
2. 問い合わせ内容をClaudeで分析（緊急度、感情、カテゴリ分類）
3. 分析結果に基づき自動で対応チームに割り当て
4. 優先度に応じた通知を送信
