# Claude Skills Library

Claude / ChatGPT / AI Automation で使う  
Skills・Prompt・Workflowを管理するリポジトリ。

## Skills
- X投稿生成 (`skills/x-post-generator.md`)
- AIニュース要約 (`skills/ai-news-summary.md`)
- 日次振り返り (`skills/daily-reflection.md`)

## Workflows
- AIニュース → 要約 → X投稿 (`workflows/ai-news-to-x-post.md`)
- Claude × Make automation (`workflows/claude-make-automation.md`)

## Structure
claude-skill
├ skills          # スキル定義ファイル
├ workflows       # ワークフロー定義
├ templates       # 作成テンプレート
└ docs            # 使用ガイド
