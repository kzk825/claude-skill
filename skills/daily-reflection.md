# 日次振り返りスキル

## 概要
1日の活動を振り返り、学びや改善点を整理するスキルです。生産性向上と継続的な成長をサポートします。

## 入力
- 今日の活動: 行ったタスクやプロジェクト
- 成果: 達成したこと
- 課題: 直面した問題や困難
- 感情: 今日の全体的な感情（ポジティブ/ネガティブ/ニュートラル）

## 出力
- 振り返りサマリー: 今日の全体的な評価
- 学び: 得られた教訓や知見
- 改善点: 次に活かせる改善策
- 明日への目標: 次の日の行動計画

## 使用例
今日の活動: AIプロジェクトの開発、会議参加
成果: 新機能の実装完了
課題: 技術的なバグの解決に時間がかかった
感情: ポジティブ（達成感がある）

## プロンプト
```
あなたは生産性と自己成長の専門家です。以下の情報を基に、日次振り返りを行ってください：

今日の活動: {今日の活動}
成果: {成果}
課題: {課題}
感情: {感情}

以下の構造で振り返りを作成してください：

1. 今日の評価（5段階評価と理由）
2. 主要な成果とその意義
3. 直面した課題と解決策
4. 得られた学び
5. 改善点と行動計画
6. 明日への抱負

建設的な視点で、バランスの取れた振り返りを日本語で記述してください。
```
