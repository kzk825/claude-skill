# X投稿生成スキル

## 概要
X（旧Twitter）への投稿コンテンツを生成するスキルです。魅力的な投稿文を作成し、ハッシュタグや絵文字を適切に使用します。

## 入力
- トピック: 投稿したい内容のテーマやキーワード
- トーン: 投稿のトーン（例: ポジティブ、ユーモラス、真剣）
- 文字数制限: 希望する文字数（デフォルト: 280文字以内）

## 出力
- 投稿文: Xに投稿可能な形式のテキスト
- ハッシュタグ: 関連するハッシュタグの提案
- 投稿理由: この投稿が効果的な理由

## 使用例
トピック: AI技術の進歩
トーン: 教育的なポジティブ

## プロンプト
```
あなたはX（旧Twitter）の投稿作成の専門家です。以下の条件で魅力的な投稿を作成してください：

トピック: {トピック}
トーン: {トーン}
文字数制限: {文字数制限}

以下の要素を含めてください：
1. 読者の興味を引く導入部
2. 主要な情報や洞察
3. 行動喚起（いいね、リポスト、コメントを促す）
4. 関連するハッシュタグ（2-3個）
5. 適切な絵文字

投稿は日本語で、Xの文字数制限内に収まるようにしてください。
```
