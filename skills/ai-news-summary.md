# AIニュース要約スキル

## 概要
AI関連のニュース記事を読み取り、重要なポイントを簡潔に要約するスキルです。技術的な詳細と影響をバランスよくまとめます。

## 入力
- ニュース記事: 要約したい記事の本文またはURL
- 要約長: 要約の長さ（短い/中間/長い）
- 重点: 注目したいポイント（技術詳細/影響/将来展望など）

## 出力
- 要約文: 記事の主要ポイントをまとめたテキスト
- キーポイント: 箇点でまとめた重要なポイント
- 影響度: このニュースの業界への影響度（高/中/低）
- 関連キーワード: 関連する技術用語やキーワード

## 使用例
ニュース記事: OpenAIの新モデルGPT-5発表
要約長: 中間
重点: 技術詳細と影響

## プロンプト
```
あなたはAIニュースの専門アナリストです。以下のニュース記事を分析し、要約してください：

記事内容: {ニュース記事}
要約長: {要約長}
重点: {重点}

以下の構造で要約を作成してください：

1. ヘッドライン（簡潔な見出し）
2. 要約（{要約長}程度の長さ）
3. キーポイント（3-5個の箇点）
4. 業界への影響
5. 関連キーワード

技術的な正確性を保ちつつ、わかりやすい日本語で記述してください。
```
